# Filename Sanitizer - 구현 계획서

## 📋 개요

사용자가 입력한 텍스트를 운영체제에서 유효한 파일 이름으로 변환하는 도구입니다.
OS별 허용되지 않는 문자를 필터링하고, 다양한 부가 옵션을 제공합니다.

### 핵심 요구사항

- 운영체제(Windows/Linux/macOS)별 금지 문자 표시 및 필터링
- 기본값: 세 OS 모두에서 사용 가능한 문자만 허용 (가장 엄격한 모드)
- 사용자가 OS를 개별로 켜고 끌 수 있음
- 부가 기능은 기본적으로 숨기고 점진적 공개 (Progressive Disclosure)

---

## 🔤 OS별 금지 문자

### Windows

- 금지 문자: `\ / : * ? " < > |`
- 제어 문자: 0x00 ~ 0x1F
- 예약된 이름: CON, PRN, AUX, NUL, COM0-COM9, LPT0-LPT9
- 끝에 `.` 또는 공백 불가

### Linux

- 금지 문자: `/`
- 제어 문자: 0x00 (NUL)

### macOS

- 금지 문자: `/`, `:`
- 제어 문자: 0x00 (NUL)
